# SG Bank Web Application

SG Bank is a simple banking web application built using **Python (Flask)**.  
The project simulates core banking functionalities such as user authentication, balance viewing, deposits, withdrawals, and transaction history management.

This project was developed as a **learning and portfolio project** to demonstrate backend web development concepts, authentication, and basic financial logic.

---

## Features

- User registration and login system
- Secure session-based authentication
- Dashboard displaying account balance
- Deposit and withdraw funds
- View transaction history
- Delete transactions
- Clean and responsive UI using Bootstrap
- Simple and clear project structure

---

## Installation & Setup

1. Clone the repository
-git clone https://github.com/Sbonga74/sg-bank-web-app.git

2. Navigate into the project directory
-cd sg-bank-flask-app

3. Create and activate a virtual environment
-python3 -m venv venv
-source venv/bin/activate

4. Install dependencies
-pip install flask

5. Run the application
-python app.py


6. Open your browser
-http://127.0.0.1:5000
